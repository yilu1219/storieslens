# Visual Write Co-Creation Design

## Goal

Make classroom co-creation the primary Visual Write experience while preserving free writing as an accessible secondary mode.

## Entry Experience

The default page presents two choices. Join Class Project is the larger primary card with class code and student name fields, a live project preview, and progress. Free Writing is the quieter secondary card. Assignment URLs with `mode=assignment&code=...` bypass the choice and open the task directly.

## Assignment Experience

The writing workspace keeps the student's assigned scene or chapter editable and shows the whole-class project context: project title, output type, class progress, chapter/scene authors, status, the student's current part, and the publish goal. Students can view peers' status but cannot edit peers' writing.

## Free Experience

Free Writing retains grade selection, idea prompt, writing editor, AI feedback, image/video generation, and local save.

## Visual Direction

Use the existing star field and glass panels. The co-creation card receives stronger scale, brighter blue emphasis, and a visible class progress strip. Free Writing remains visually available without competing with the core classroom path.
